### 《职业星图》播客

15年猎头生涯，深耕互联网与AI行业，见证了许多人职业旅程的起伏与蜕变。也愈发相信，每一个真实的职业故事，都拥有照亮他人的力量。

个体的职业史，就是时代的星图。而我一直在寻找每一颗星光的意义，试图从无数个体的选择与坚持中，拼凑出属于这个时代的职业脉络与精神图景。

因此，我发起了一个名为《职业星图》的自媒体项目（播客/视频采访形式），旨在通过采访不同行业、领域的优秀职场人士，还原从「专业」到「职业」的成长路径，呈现每一次抉择背后的思考与蜕变，为大学生、求职者和转型人士提供有价值的参考。

欢迎感兴趣的朋友交流👏。

如果你有故事，或者你认识有故事的人，欢迎自荐或引荐，让好故事讲给更多人听到！
